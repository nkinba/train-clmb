# docs/design/

Stitch에서 생성한 화면 mock 보관 디렉토리.

## 파일 규칙
- 화면 mock은 PNG 또는 SVG로 저장.
- 파일명은 `STITCH_PROMPTS.md`에서 지정한 이름을 사용:
  - `home-today.png`
  - `hangboard-timer.png`
  - `set-result.png`
  - `bouldering-quick-input.png`
  - `analysis.png`
- 반복 작업 시 버전 접미 사용: `home-today-v2.png` 등.
